# whichmovie

一个用来帮助选择影片观看的工具脚本，数据来自豆瓣电影。

脚本的目的是练习使用python语言，涉及如下内容练习：

1. 列表推导式
2. 简单类
3. urllib网络请求
4. json序列化为dict、自定义class对象
5. 异常机制
6. venv虚拟环境
